## 快捷键 mac

- 代码跳回： option + command + left 
- 查看某个方法的调用链： control + option + h  (需要把光标定位到某个方法上)
- 查看类或接口包括其子类的继承关系：ctrl + h
- 显示该类、方法被哪里调用：Alt+F7
- 查看某个接口的所有实现类或子类：option + command + b (需要把光标定位到某个接口上)
- 全局搜索: command + shift + f
- 换行(无论是否处于行尾)：command + shift + enter
